# kwshell

A simple and cute terminal application for Android that uses Shizuku to execute commands.

## Features

*   **Shizuku-powered:** Executes commands with elevated privileges using the Shizuku service.
*   **Kawaii UI:** A cute, pink-themed interface for a more enjoyable terminal experience.
*   **Gatekeeper:** A modern, integrated gatekeeper that ensures Shizuku is running before allowing access to the terminal.

## Getting Started

To build and run this project, you will need:

*   Android Studio
*   An Android device or emulator with Shizuku installed and running

Simply open the project in Android Studio and run the `app` configuration.
